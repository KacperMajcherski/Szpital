INSTRUKCJA:
1. Otwórz VISUAL STUDIO CODE
2. Przełącz się na brancha MASTER git checkout master
3. Update brancha ze zdalnego repo: git pull
4. Stwórz nowego brancha git checkout -b <nazwa_brancha> , na przykład: git checkout 06.01.2026
5. Zrób swoje zmiany
6. ZAPISZ zmiany w Visual Studio
7. Git add . , potem git commit -m "nazwa_commita"
8. git push --set-upstream origin <nazwa_brancha>
9. Czekasz na pull request i merge do MASTER

Niniejszy projekt jest systemem administracyjnym dla szpitala. Język, w którym jest napisany, to C#. System kontroli wersji, który został użyty do zarządzania repozytorium kodu, to Git. Użyto również GitHub jako miejsce do przechowywania kodu w chmurze.
